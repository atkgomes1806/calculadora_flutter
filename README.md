# Calculadora Flutter

Projeto simples de calculadora desenvolvido em Dart/Flutter para a disciplina de Programacao para Dispositivos Moveis (UNIFACOL).

## Repositorio

https://github.com/atkgomes1806/calculadora_flutter.git

## Tecnologias

- Dart
- Flutter

## Como executar

1. Instale o Flutter SDK.
2. No terminal, entre na pasta do projeto.
3. Execute os comandos abaixo:

```bash
flutter pub get
flutter run
```
